# iNotes 📝

iNotes adalah aplikasi catatan sederhana berbasis website yang dirancang agar sangat responsif (mobile-friendly), interaktif, dan aman. Aplikasi ini mempermudah pengguna untuk mengelola catatan harian mereka secara real-time langsung dari smartphone maupun komputer.

---

## 🚀 Fitur Utama

*   **Google Sign-In & Registrasi Normal**: Autentikasi mudah menggunakan akun Google atau pendaftaran akun manual.
*   **CRUD Interaktif (Tanpa Reload)**: Menambah, mengedit, dan menghapus catatan secara instan berkat integrasi JavaScript (Fetch API).
*   **Sematkan Catatan (Pin)**: Menjaga catatan penting agar tetap berada di baris paling atas.
*   **Kustomisasi Warna Kreatif**: Pengguna bisa memilih warna latar belakang catatan mereka.
*   **Auto-Contrast Text**: Teks otomatis menyesuaikan warna (gelap/terang) bergantung pada warna latar belakang yang dipilih agar tetap nyaman dibaca.
*   **Toast Notifications**: Notifikasi melayang yang interaktif saat catatan berhasil disimpan, diubah, atau dihapus.
*   **Responsive Web Design**: Tampilan mobile-first yang rapi menggunakan Tailwind CSS.

---

## 🛠️ Teknologi yang Digunakan

*   **Backend**: PHP Native (OOP/Procedural clean style) & MySQL
*   **Frontend**: HTML5, Tailwind CSS (via CDN), JavaScript (ES6 & Fetch API)
*   **API**: Google Identity Services (OAuth 2.0)
